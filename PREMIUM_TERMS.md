# Weave EPUB Reader Premium Terms Overview

本文件用于说明 Weave EPUB Reader 的“基础免费 + 高级付费”模式边界，帮助用户理解高级能力、激活机制与开源代码许可之间的关系。

## 1. 模式概述

Weave EPUB Reader 采用以下模式：

- 基础功能免费
- 高级功能付费
- 仓库公开源码继续遵循开源许可证

这意味着：

- 并非所有功能都需要付费
- 付费主要对应高级能力、许可证服务、在线能力或持续维护相关附加价值

## 2. 基础免费功能

基础免费功能通常包括：

- 本地阅读（**EPUB** 与 **TXT** 格式）
- 书架管理
- 阅读设置
- 基础摘录、高亮、回链与来源定位
- 与 Markdown、Canvas、Weave 数据的核心本地联动

具体以实际版本发布时的功能状态为准。

## 3. 高级功能

高级功能通常指：

- 需要激活后才能使用的能力
- 更完整的高级阅读工作流
- 更强的集成能力
- 可能依赖在线许可证服务的功能

具体高级功能集合可能随着版本更新调整，以插件界面、发布说明与购买说明为准。

## 4. 激活与许可证服务

某些高级能力可能依赖以下机制：

- 激活码
- 绑定邮箱
- 许可证验证
- 设备额度管理
- 在线状态校验

如果高级能力依赖这些机制，用户需要接受相应的授权与服务流程。

## 5. 开源协议与付费的关系

当前仓库代码按 GPL-3.0-or-later 发布。

这表示：

- 公开发布的源码部分继续受 GPL 约束
- 付费模式不会抹除 GPL 对已发布源码赋予用户的权利
- 高级版收费主要针对产品分层、服务能力、在线能力、许可证体系与持续维护

如果某项高级能力依赖在线服务、账号体系或许可证基础设施，其使用条件也可能由额外说明补充。

## 6. 用户应知晓的边界

- 购买高级版不等于购买对 Vault 数据的所有权变更
- 用户自己的 EPUB、Markdown、Canvas 与 Vault 数据仍由用户自己控制
- AI 功能通常还需要用户自行提供第三方 AI 服务的 API Key
- 某些高级能力可能因许可证状态、设备额度或在线服务可用性而受到影响
