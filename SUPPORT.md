# Weave EPUB Reader Support

本文件用于说明 Weave EPUB Reader 的公开反馈渠道，以及不同类型问题建议的处理方式。

## 适合在 GitHub 反馈的问题

如果你遇到以下问题，适合优先通过 GitHub Issues 反馈：

- 阅读器打不开某类书籍文件
- 阅读进度、书签、目录跳转异常
- 高亮、摘录、回链、导出等本地功能异常
- 与 Markdown、Canvas、Weave 数据联动相关的问题
- 文档、README、安装说明不清晰

GitHub:

- https://github.com/zhuzhige123/obsidian-weave-reader

## 激活、许可证与高级版相关问题

如果你遇到以下问题：

- 激活失败
- 邮箱绑定异常
- 设备数量或设备迁移问题
- 高级版权限未正确生效

建议在反馈时尽量提供：

- 插件版本号
- 使用平台（Windows / macOS / Linux / Android / iOS）
- 报错提示文本
- 是否已更换设备或重装系统

为避免泄露敏感信息：

- 不要公开贴出完整激活码
- 不要公开贴出完整邮箱验证码或其他私密凭证
- 如必须展示，请只保留必要片段并进行打码

## AI 功能相关问题

如果问题与 AI 能力有关，建议区分两类：

- 插件侧问题：按钮、面板、配置读写、调用流程、错误提示异常
- 第三方服务问题：API Key 无效、模型不可用、额度不足、服务商超时

反馈时建议说明：

- 使用的 AI 服务提供商
- 是否使用自定义 Base URL
- 插件报错信息
- 是否仅某个模型出错

请不要公开贴出你的 API Key。

## 不建议在公开 Issue 中直接提交的内容

以下内容不建议直接公开发布在 Issue、评论或截图中：

- 完整激活码
- 邮箱验证码
- 私钥、令牌、API Key
- 包含私人阅读内容的完整截图
- 含敏感 Vault 路径或私人文件名的日志

## 反馈前建议先确认

- 已升级到最新公开版本
- 已阅读 [README.md](./README.md)
- 已阅读 [PRIVACY.md](./PRIVACY.md)
- 已阅读 [PREMIUM_TERMS.md](./PREMIUM_TERMS.md)

## 说明

本文件是公开支持说明，不保证固定响应时限，也不构成单独服务承诺。若未来提供更正式的支持流程，以届时的公开说明为准。
